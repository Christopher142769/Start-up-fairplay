Dossier médias Start-up Fair Play
================================

Emplacement : client/public/branding/

Fichier attendu par l’application :
  logo.png   → logo principal (fond transparent de préférence)

URL en production / dev : /branding/logo.png

Pour remplacer le logo : écrasez logo.png en conservant le même nom de fichier.

Performance : pour un chargement rapide, viser une image légère (idéalement WebP
logo.webp sous ~200 Ko, ou PNG optimisé). Les très gros fichiers ralentissent
l’affichage même avec mise en cache.
